# TelegramTradingBot
 A Telegram trading bot

## Installation Instructions

### Clone the repo
```bash
git clone https://github.com/A-Unix/TelegramTradingBot.git
```
### Move to cloned folder
```bash
cd TelegramTradingBot
```
### Get permissions
```bash
chmod +x *
```
### Install dependencies
```bash
./setup.sh
```
### Run the main file
```bash
python3 main.py
```
Enjoy :)
